Networking Projects (Cisco Packet Tracer)
Overview

This repository contains a collection of hands-on networking labs focused on practical, real-world networking concepts commonly used in enterprise IT environments.

The projects emphasize Layer 2 and Layer 3 networking fundamentals, including switching, routing, VLAN segmentation, and traffic verification. All labs are designed to reinforce understanding of how data moves through a network and how configuration decisions impact connectivity.

Each project is fully documented and includes configuration artifacts and validation steps.

Repository Structure

Each project is contained within its own folder and includes:

A fully configured Cisco Packet Tracer (.pkt) file

A labeled topology diagram or supporting screenshots

A project-specific README.md detailing design, configuration, and testing
