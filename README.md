# Networking Projects

This repository contains hands-on networking labs designed to demonstrate practical skills in switching, routing, VLANs, and enterprise network design using Cisco Packet Tracer.

Each project includes a fully configured Packet Tracer file, a labeled topology diagram, and documentation outlining design decisions, configuration steps, and verification.
